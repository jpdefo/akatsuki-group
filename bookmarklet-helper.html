Akatsuki Live Sync Helper

This page makes the live SteamGifts sync easier to run from your logged-in browser.

  1. Keep python server.py running.
  2. Open the Akatsuki SteamGifts group in Chrome.
  3. Copy the bookmarklet below.
  4. Type javascript: in the address bar.
  5. Paste the copied body after it and press Enter.
  6. It starts from page 1 and only asks how many giveaway pages to fetch.
  7. Return to the dashboard and click Load server sync.
Open SteamGifts group Open dashboard

Chrome may reject the full javascript: form when pasted directly, so this copies only the body.

The local server now merges batches, so you can safely run page 1, then page 2, then page 3 over time instead of scraping too much at once.

The default delay is now fixed at 505ms between requests.
